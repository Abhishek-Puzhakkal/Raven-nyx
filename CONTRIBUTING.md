# RAVEN NYX

## INTRODUCTION

Ravenyx is a privacy-focused peer-to-peer messaging and file-sharing system that supports encrypted one-to-one and group communication over Tor onion services and LAN. It uses the Noise Protocol Framework (NN handshake) to establish secure sessions without relying on centralized servers

## Getting Started with Contributions

If you are new to open source, you can learn the basic contribution workflow from below mentioned repo:

[first-contributions](https://github.com/firstcontributions/first-contributions)

This project is also meant to be beginners or the curious peoples out there , so don't hesitate to ask questions or suggest improvements.

You can contribute in several ways:

- Fixing bugs
- Improving documentation
- Refactoring code for readability
- Improving error handling
- Adding small features
- Improving Tor connection stability
- Suggesting new ideas


Also communicate before commiting you time on contribution's, or working on any issues , because may some other may also work on same thing , so to avoid conflicts communicate before starting work on an issue


Typical contribution process:

1. Fork the repository
2. Clone your fork
3. Create a new branch for your change
4. Make your changes
5. Test your changes
6. Submit a Pull Request

Please include a clear description of what your change does.
